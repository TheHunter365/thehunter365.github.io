Vertical Dash

Welcome to Vertical Dash!

Vertical Dash is a simple game where you control a blue ship using the space bar. The objective is to avoid red obstacles and score as many points as possible. The game speeds up as your score increases.

How to Play:

  • Press the space bar to make the ship ascend or descend.
  • Avoid colliding with the red obstacles.
  • Your score increases with time, and the game speeds up.
  • The game ends if the ship collides with an obstacle.
  • Press the space bar to restart after a game over.

How This Game Was Made

This game is a school project for the Faculty of Science of Rabat, and it provides an opportunity to showcase how the game was developed.

The core of the game relies on the <canvas> element in HTML. The canvas element allows you to draw graphics and create interactive elements using JavaScript. Learn more about HTML5 canvas here.
